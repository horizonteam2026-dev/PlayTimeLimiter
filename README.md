# 🛡️ TimeGuardian - Parental Screen Time Limiter

A simple, lightweight Windows desktop application that helps parents manage and limit screen time for children with a secure full-screen lock.

## ✨ Key Features
- **Elderly-Friendly:** Large text and prominent buttons, perfect for older parents.
- **Bilingual Support:** Easily switch between English and Vietnamese with one click.
- **Strict Lock Screen:** When time runs out, a full-screen overlay blocks all inputs (Disables `Alt+F4` or closing shortcuts).
- **Pre-lock Reminders:** Displays a quick warning alert at 5 minutes and 1 minute remaining.
- **Custom Password:** Parents can easily change the unlock password directly from the menu.

## 🚀 How to Download and Use
1. Go to the **Releases** section on the right side of this page.
2. Download the latest `TimeGuardian.exe` file.
3. Open the application, enter the allowed playtime in minutes, and click **START COUNTDOWN**.
4. Once time is up, enter the parent password (Default is `1234`) to unlock the computer.
